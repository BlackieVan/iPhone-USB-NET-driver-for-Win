# iPhone USB & NET Drivers for Windows

🚀 **一个只有几 MB 的纯净驱动包，让断网的 Windows 电脑直接通过 USB 共享 iPhone 网络。**


---

## 💡 项目背景与痛点

在电脑**没有 Wi-Fi、没有网络、没装 iTunes**，但你手里只有**一台 iPhone** 和**一条 USB 数据线**的极端情况下：
* ❓ 想通过 iPhone 的 USB 共享网络让电脑上网？
* ❓ 想把 iPhone 里的照片和文件拷贝到电脑？
* ❌ 官方 iTunes 安装包动辄几百 MB，且在没网的情况下根本无法下载安装。

**本项目正是为了解决这个“鸡生蛋、蛋生鸡”的断网死循环而生。**

---

## ✨ 核心特性

* 📦 **极限精炼**：完全剔除官方 iTunes 软件全家桶，只提取最核心的驱动文件（不到 3MB）。
* 👥 **双架构支持**：同时完美集成 **x64（64位）** 与 **x86（32位）** 核心驱动，通杀从老旧 Win7 到现代 Win11 的绝大多数机型。
* 🔒 **纯净无后台**：纯手动设备管理器强刷，没有任何后台常驻服务、无开机自启，绿色环保。
* 🗺️ **套娃黑科技**：支持将驱动隐写伪装在 JPG 图片中，直接用手机相册当“U盘”安全带出。

---

## 🛠️ 隐写术原理（怎么做到的？）

利用 Windows 的二进制合并命令，将驱动压缩包藏在图片数据末尾，生成一张在手机和电脑上都能正常预览、但暗藏玄机的“套娃图片”：
```bash
copy /b iPhone-USB&NET-driver-for-Win.jpg + iPhone-USB&NET-driver-for-Win.zip iPhone-USB&NET-driver-for-Win.jpg
```

- **在手机端**：它是一张普通的物理照片，可以正常保存到相册、通过 iOS 协议被电脑读取。
    
- **在电脑端**：将其拷贝出来改后缀为 `.zip`，即可用解压软件直接释放出真正的驱动程序。
    

## 🚀 使用指南（怎么用？）

### 第一步：从手机中提取驱动

1. 将项目中的 `iPhone-USB&NET-driver-for-Win.jpg` 保存到你的 iPhone 手机相册中。
    
2. 使用 USB 数据线将 iPhone 连接到断网的电脑，并在手机上弹出提示时点击 **「信任此电脑」**。
    
3. 打开电脑的“此电脑/资源管理器”，进入 `Apple iPhone` 便携式设备，在照片文件夹（DCIM）里找到这张照片。
    
4. 将照片拷贝到电脑桌面，并将文件后缀名直接改为 `.zip`。
    
5. 使用 7-Zip 或 WinRAR 打开该压缩包，将其中的驱动文件夹解压出来。
    

### 第二步：开启手机热点

- 在 iPhone 上打开 **「设置」 -> 「个人热点」** -> 开启 **「允许其他人加入」** 开关。
    

## 🔧 驱动手动安装方法

根据你的 Windows 系统版本，选择对应的安装方法：

### 🖥️ 方法 A：在 Windows 10 / Windows 11 上安装

**第 1 阶段：标准自动检索安装（首选）**
1. 右键点击 Windows 开始菜单，选择 **「设备管理器」**。
2. 在列表中找到带有 **黄色感叹号** 的未知设备（通常在“其他设备”或“便携设备”分类下，硬件 名称包含 `Apple`字样）。
3. 右键点击该设备 -> 选择 **「更新驱动程序」** -> 点击 **「浏览我的电脑以查找驱动程序」**。
4. 点击「浏览」按钮，直接选中你解压出来的驱动文件夹（根据系统架构进入 `x64` 或 `x86` 目录），确保勾选“包含子文件夹”，点击下一步。此时系统会自动识别 `netaapl.inf` 并加载 `WdfCoInstaller01009.dll` 框架。如果顺利，设备会自动归类并完成联网。

**第 2 阶段：强制指定分类安装（仅当安装后出现“问号”时执行）**
1. 如果执行完上述步骤后，设备图标变成了**问号**（或仍显示为带问号的未知设备），请再次右键点击该问号设备 -> 选择 **「更新驱动程序」** -> **「浏览我的电脑...」**。
2. 这一次点击下方的 **「让我从计算机上的可用驱动程序列表中选取」**（关键步骤）。
3. 在弹出的硬件类别列表中，死死往下翻，找到并双击 **「网络适配器」 (Network adapters)**，然后点击下一步。
4. 点击右下角的 **「从磁盘安装」** -> 点击 **「浏览」** -> 精准锁定你 U 盘目录下的 **`netaapl.inf`**，点击确定。
5. 在型号列表里选中强制刷出的 `Apple Mobile Device Ethernet`，点击下一步。如果系统弹出未签名警告，直接选择 **「始终安装此驱动程序」** 即可强行激活网络。
    

### 💾 方法 B：在 Windows 7 上安装

1. 打开 Windows 设备管理器，在 **「便携设备」** 下面找到 **"Apple iPhone"**。
    
2. 点击鼠标右键，依次选择：
    
    - **「更新驱动程序软件」**
        
    - **「浏览计算机以查找驱动程序软件」**
        
    - **「从计算机的设备驱动程序列表中选择」**
        
    - **「从磁盘安装」**
        
3. 点击 **「浏览」**，然后找到刚才解压出来的 **`usbaapl.inf`**（若为64位系统则选择 `usbaapl64.inf`），选中它进行安装。
